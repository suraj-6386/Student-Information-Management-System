ISSUE: "Connection refused" when accessing database
SOLUTION:
  - Start MySQL: Data\mysql\bin\mysqld.exe
  - Or: Start MySQL from Services (Windows)

ISSUE: Database tables don't exist
SOLUTION:
  - Open DBquery.txt
  - Copy all commands
  - Execute in MySQL Workbench
  - Check for error messages

ISSUE: Page shows 404 error
SOLUTION:
  - Check Tomcat is running (look for Tomcat icon in taskbar)
  - Verify URL: http://localhost:8080/MyApps/StudentInfoManageSystem/
  - Clear browser cache (Ctrl+Shift+Delete)

ISSUE: Login fails
SOLUTION:
  - Username and password are CASE-SENSITIVE
  - Check: admin (not Admin), teacher, student
  - Verify database has login table

ISSUE: Student dashboard shows error
SOLUTION:
  - Ensure student was added from admin panel
  - Restart Tomcat
  - Clear session: logout and login again
